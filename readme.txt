Animated typing text widget for Elementor powered by Typed.js

== Description ==

Typed.js Elementor Widget adds a lightweight animated typing widget to Elementor, powered by the popular Typed.js library.

Create engaging typewriter-style text directly inside the Elementor editor, without writing custom code. Add multiple text lines, adjust typing and backspacing speed, control delays, enable looping, and choose whether the animation should fade out or delete characters.

The widget also includes dedicated cursor and typography controls, so you can fine-tune the cursor character, cursor color, text color, alignment, font family, font weight, font size, and line height from Elementor's visual interface.

It is designed to be simple, flexible, and easy to style, making it useful for hero sections, headings, landing pages, service pages, and any area where animated text can help highlight key messages.

= Why I built this plugin =

I built this plugin after experimenting with Typed.js in Elementor layouts and realizing that the same typewriter-style setup could be useful as a reusable widget. I have always found the typewriter effect to be a nice way to add movement and emphasis to a page, and turning it into a plugin makes it easier to configure without writing custom code each time.

This plugin is my way of making that effect easier to reuse in Elementor projects, both for myself and for anyone else who may find it useful. I hope it helps other creators add a polished animated text effect to their websites more quickly.

Typed.js itself is developed by Matt Boldt. Many thanks to Matt for creating and maintaining such a great library.

= Main features =

* Native Elementor widget
* Multiple animated text lines
* Typing, backspacing, start delay, and back delay controls
* Loop and fade-out options
* Custom cursor character and cursor color
* Text color, alignment, font, weight, size, and line-height controls
* Responsive font size settings
* Local Typed.js asset included, with no external CDN dependency

== Plugin Dependencies ==

This plugin requires Elementor.

== Installation ==

1. Copy the `typed-elementor-widget/` folder into `/wp-content/plugins/`
2. Activate the plugin from the WordPress Plugins panel
3. Open Elementor and search for the widget "Typed.js Animator" in the left sidebar
